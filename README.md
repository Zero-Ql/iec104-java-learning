### IEC 104 Master Station Implementation (GPL-3.0)
https://www.gnu.org/licenses/gpl-3.0

基于Netty的IEC 60870-5-104规约主站端Java实现。
### 已实现功能
- 启动帧、总召唤发送
- 解析总召响应、遥测短浮点、遥信单点I帧
- U测试帧与S帧自动维护
- 其他功能开发中
